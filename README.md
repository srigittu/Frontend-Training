# Frontend-Training
Training in angular js

## Procedure to setup:

1.Clone git repo 

```
	git clone https://github.com/srigittu/Frontend-Training.git
```

2.Navigate to app folder 

```
	cd Frontend-Training
```

3.Fetch repo 

```
	git fetch origin
```

4.Switch branch to todo 

```
	git checkout todo
```

5.Navigate to dev folder

```
	cd dev
```

6.Install node modules 

```
	npm install
```

7.Run app 

```
	gulp
```
